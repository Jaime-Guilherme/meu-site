# Portfólio — Jaime Guilherme

Este é um site simples de portfólio/serviços para Jaime Guilherme (técnico em eletrônica e informática).

Como usar
1. Abra `index.html` no navegador para testar localmente.
2. Edite o conteúdo (sobre, serviços, projetos) diretamente no `index.html`.
3. Atualize as imagens em `medicao-de-quadro.jpg` e `imagem-de-fundo-2.jpg` — mantenha os nomes ou atualize os `src` no HTML.

Arquivos importantes
- `index.html` — markup principal (header, serviços, projetos, contato).
- `estilo.css` — estilos do site.
- `medicao-de-quadro.jpg`, `imagem-de-fundo-2.jpg` — imagens usadas no site.

Melhorias simples que você pode aplicar
- Substituir imagens por versões otimizadas (WebP) e redimensionadas para web.
- Editar os links em `#projetos` para apontarem para páginas ou repositórios reais.
- Adicionar `cv.pdf` na raiz se quiser disponibilizar o currículo para download.

Deploy rápido no GitHub Pages
1. Crie um repositório no GitHub e faça `git init`/`git add .`/`git commit -m "site"`.
2. Faça `git remote add origin <URL-do-repo>` e `git push -u origin main`.
3. No repositório GitHub vá em Settings → Pages e selecione a branch `main` (root) para publicar.

Observações
- Verifique que os arquivos de imagem existem na pasta; se mudar nomes, atualize os `src` em `index.html`.
- Se quiser, posso gerar um formulário de contato (Formspree) ou preparar o site para deploy automático no Netlify/Vercel.

---
Se desejar, aplico mais melhorias: otimização de imagens, geração de miniaturas, criação de seção detalhada de projetos com modal e integração com formulário de contato.
